# Pinwheel

Pinwheel is a TypeScript library for animating SVG icons in React.

## Features

- 12 built-in icon animations (rotate, fade, bounce, scale)
- Sub-100ms render budget
- Tree-shakeable; importing one icon pulls in less than 4KB gzipped
- Compatible with React 18 and React 19

## Installation

npm install pinwheel-icons
