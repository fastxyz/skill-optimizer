# MCP Tracker Demo Mock Repo

This template is a richer MCP optimizer target for issue-tracker style workflows.

The tool schema in `tools.json` is intentionally awkward, so benchmark quality depends heavily on `SKILL.md`.

Use these files together:
- `benchmark.config.json` for benchmark runs
- `optimize.config.json` for optimizer runs

During optimization, only `SKILL.md` should be changed.
